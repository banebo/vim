# vim
vim configuration with plugins

1st run:
    sudo apt-get install vim-pathogen -y
    cp -r ~/vim/.vim ~/.vim
    cp ~/vim.vimrc ~/.vimrc

Commands:
    Ctrl + n         -> NERDTree
    f8               -> run ALEFix (fixes the file format)
    Ctrl + (h,j,k,l) -> buffer navigation
    space            -> folds functions
